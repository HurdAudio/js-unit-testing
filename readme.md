#Javascript Unit Testing

If you haven't already, globally install Mocha:
```
$ npm install mocha -g
```

Next, build out a directory that is ready for testing:
```
$ mkdir js-unit-testing
$ cd js-unit-testing
$ git init
$ npm init -y
$ npm install --save-dev chai
$ touch main.js
$ mkdir test
$ cd test
$ touch test.js
```
